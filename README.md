# MSBI Project – SSMS, SSIS, SSRS

## 📌 Project Overview
This project demonstrates the use of **Microsoft Business Intelligence (MSBI)** tools to design, build, and present a data warehouse solution.  
It follows a **star schema model** with one central fact table (`FactCustomer`) and multiple dimension tables (`Customer`, `Country`, `States`, `Product`, `SalesPerson`).  

The workflow covers:
- **SSMS (SQL Server Management Studio)** → Database creation, schema design, and sample data insertion.
- **SSIS (SQL Server Integration Services)** → ETL (Extract, Transform, Load) processes to move and transform data.
- **SSRS (SQL Server Reporting Services)** → Reports and dashboards for business insights.
